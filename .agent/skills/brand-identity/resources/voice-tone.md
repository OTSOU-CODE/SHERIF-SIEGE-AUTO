# Copywriting: Voice & Tone Guidelines

When generating text, adhere to this brand persona.

## Brand Personality Keywords

- **Luxury & Premium**: "Bespoke", "Refined", "Exclusive".
- **Professional but approachable**: Knowledgeable about cars/upholstery but welcoming.
- **Direct and efficient**: No fluff.
- **Empathetic**: Understanding the customer's desire for quality.

## Grammar & Mechanics rules

- **Headings:** Use Title Case for main headings (H1, H2). Use sentence case for subheadings (H3+).
- **Punctuation:** Avoid exclamation points (!). Use periods for complete sentences.
- **Clarity:** Prefer active voice over passive voice. Keep sentences concise.

## Terminology Guide

| Do Not Use       | Use Instead                 |
| :--------------- | :-------------------------- |
| "Cheap"          | "Affordable", "Competitive" |
| "Stuff"          | "Materials", "Components"   |
| "Utilize"        | "Use"                       |
| "In order to..." | "To..."                     |
