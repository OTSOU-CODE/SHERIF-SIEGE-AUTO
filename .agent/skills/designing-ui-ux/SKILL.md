# Modern Design & UX Excellence Framework

This skill provides a rigorous standard for creating world-class web experiences. It covers everything from visual polish to technical performance.

## When to use this skill

- When designing a new feature or page.
- When conducting a UI/UX audit.
- When the user asks to make the site "look modern" or "feel premium".
- When fixing responsiveness or accessibility issues.

## Core Principles

1.  **Aesthetics are Functional**: A beautiful interface builds trust.
2.  **Performance is UX**: A slow premium site is not premium.
3.  **Accessibility is Default**: Inclusive design is better design for everyone.
